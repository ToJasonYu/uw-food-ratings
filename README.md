## ‚ú® Features

* **Authentication System:** Secure Login and Registration functionality.
* **Dynamic Feed:** Real-time list of reviews with "Upvote" capabilities.
* **Search & Filter:** Instantly filter reviews by Restaurant Name, Dish Name, or sort by Highest/Lowest ratings.
* **Glassmorphism UI:** A modern, responsive interface built with CSS3 and React.
* **CRUD Operations:** Users can Create reviews, Read the feed, Update vote counts, and Delete their own posts.
